config of emacs and vim

## usage

### download
~~~bash
$ mkdir /tmp/my-config
$ git clone https://github.com/bopjiang/my-configs.git ~/code/my-configs
$ ln -s ~/code/my-configs/emacs.d ~/.emacs.d
~~~
